# B2 — ручной чек-лист кошелька

Механика флоу проверяется headless-скриптом (`scripts/b2-devnet.sh`) — он
гоняет тот же код `miniapp/src/flows.ts` файловыми ключами. Руками остаётся
то, что скрипту недоступно: браузерный кошелёк и Telegram-транспорт.

## Подготовка

1. `scripts/b2-devnet.sh` зелёный (иначе руками проверять нечего).
2. Локальная реплика с канистрой подписки запущена (скрипт оставляет её,
   если запустить с `KEEP=1`, либо поднять руками из `../Crown-Games/Subscription`).
3. `npm --workspace @crown/miniapp run dev` — vite отдаёт страницу на
   `http://localhost:5173`.
4. В браузере установлен Solana-кошелёк со стандартом Wallet Standard
   (Phantom, Solflare, Backpack), переключённый на devnet, с ключом донора
   или владельца (импорт из `~/.cache/crown-e2e/*.json`).

Ссылки строятся ботом (B3); до него фрагмент собирается руками:
`#<действие>=<base64url(JSON)>` — поля по `miniapp/src/main.ts`, плюс
`subscription` (principal канистры) и `icHost` (`http://127.0.0.1:<порт>`).

## Чек-лист

- [ ] **link**: страница показывает челлендж; кошелёк подписывает; вне
      Telegram результат `{wallet, signature, nonce}` печатается на страницу;
      внутри Telegram (после B3) — уходит через `sendData` и окно закрывается.
- [ ] **subscribe**: выбор числа периодов; кошелёк донора подписывает одну
      транзакцию; страница печатает адрес эскроу; эскроу виден в эксплорере,
      баланс = chunk × n_chunks.
- [ ] **collect**: кошелёк владельца; страница находит созревшие куски,
      **одно подтверждение на пачку**; выплаты видны в эксплорере
      (97% куска владельцу, `Settled` от сплиттера).
- [ ] **collect** при двух созревших кусках одного эскроу: оба выпускаются,
      порядок индексов строгий (транзакции уходят последовательно).
- [ ] **cancel**: кошелёк донора подписывает авторизацию (§8) — сообщение
      читаемо в кошельке как байты; остаток возвращается мгновенно; эскроу
      терминален; повторная отмена даёт ошибку формы.
- [ ] Страница не делает ни одного запроса к сервису бота (вкладка Network:
      только RPC и `icp-api`/локальная реплика).
- [ ] `sendData`-нагрузка < 4096 байт (печатается в консоли при link).
